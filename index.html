Rotating Cube

The rotating cube project was a foundational exercise in graphics programming that introduced me to the essentials of computer graphics and rendering. In this project, I created a cube and a grid using vertices, drew lines between these vertices, and set up a camera to change the viewpoint of the cube as it spins. To draw the lines, I implemented a line-drawing algorithm that connects the vertices of the cube and the grid. This involved calculating the positions of the vertices in 3D space and projecting them onto the 2D raster window. I then used these projected coordinates to draw lines between the vertices, creating the edges of the cube and the grid. Additionally, I wrote the rendered output to the raster window, ensuring that the lines and shapes were correctly displayed. This required managing the raster buffer and updating it with the new frame data for each rotation of the cube.

The project features three modes:

Default Mode: A spinning green outline of the cube.
Depth Mode: A colored depth representation of the cube.
Texturing Mode: Utilizes UV mapping and barycentric coordinates to apply textures to the cube’s surfaces.

ChatServer is an application that was designed to test my skills in Network Programming. It functions as a server compatible with the Spaghetti Relay client application.
